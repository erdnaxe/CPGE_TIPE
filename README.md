# Mon TIPE de CPGE PSI

Le sujet de mon TIPE est sur la création d'un stabilisateur motorisé à un axe.
Le but est de comprendre et réaliser un système similaire à
[l'OSMO de DJI](https://www.dji.com/fr/osmo).
